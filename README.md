# russian-history-game

# Образовательная игра «История России»

Интерактивная 2D-игра на Godot, посвящённая ключевым этапам русской истории.

## Состав проекта
- Система эпох с историческими событиями
- Крафт и строительство
- Интерактивные исторические памятки

## Как запустить
1. Установите Godot Engine 4.x
2. Скачайте репозиторий
3. Откройте папку проекта в Godot через «Import»

Проект выполнен в рамках учебно-исследовательской деятельности.
